# Commands to use npm and install parcel
-- npm init // initialize npm
-- npm install parcel // install parcel into the project
-- npx parcel index.html // use parcel dev build
-- npx parcel build index.html // use in prod build

# What parcel does
-- Dev Build
-- Local Server
-- HMR = Hot Module Replacement
-- File Watching Alogritm - written in c++
-- Caching - Faster Builds
-- Image Optimization
-- Minification
-- Bundling - basic core of parcel
-- Compressing
-- Consistent Hashing
-- Code Splitting
-- Differential Bundling - support older browsers
-- Diagonostic
-- Error handling 
-- HTTPs
-- Tree Shaking -- removes unused code
-- Different dev and prod bundles
